# Academy Portfolio & Admin Dashboard 🎓

## 📌 Overview
A dynamic and responsive portfolio website designed for an engineering and technical academy to showcase trainees' projects. The system includes a public-facing gallery and a secured Admin Dashboard for real-time content management.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript, FontAwesome
* **Backend / BaaS:** Firebase (Authentication, Firestore Database, Cloud Storage)
* **Hosting:** GitHub Pages (Public site)

## ✨ Key Features
* **Dynamic Interactive UI:** Custom-built 3D carousel for project display and animated UI elements.
* **Admin Dashboard:** Secured route for administrators to add, view, and delete projects.
* **Real-time Database:** Integration with Firebase Firestore to instantly update the gallery without modifying the code.
* **Cloud Storage:** Direct image uploads to Firebase Storage for trainee projects.
* **Role-Based Access:** Firebase Authentication prevents unauthorized access to the control panel.
